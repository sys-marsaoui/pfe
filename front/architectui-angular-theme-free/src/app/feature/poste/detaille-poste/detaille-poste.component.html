Développeur Java - Offre d'emploi
Poste proposé :
Développeur Java
Rejoignez une équipe dynamique et contribuez à des projets innovants en utilisant des technologies Java modernes.

**Important :** Le processus de recrutement comprend un **test en ligne** sélectif. Seuls les candidats ayant réussi ce test seront considérés pour ce poste.
Responsabilités :
  • Développer des applications backend en Java.
  • Participer à la conception des systèmes distribués.
  • Collaborer avec les équipes frontend et UX.
  • Mettre en œuvre des tests unitaires et assurer le suivi des performances.
  • Travailler avec des bases de données relationnelles et NoSQL.
